🔑 기상청 API 키 발급 방법

1
data.go.kr 에 로그인 후 검색창에 "기상청_단기예보 조회서비스" 검색
2
활용신청 클릭 → 신청 (자동 승인, 즉시 사용 가능)
3
오른쪽 상단 마이페이지 → 데이터활용 → Open API → 승인된 항목 클릭 → "일반 인증키(Encoding)" 복사
4
아래 입력창에 붙여넣기 → 저장

✅ 올바른 키 형태 (둘 다 정상)

형태 1 (개인 API 인증키):

210416df60ffc822b1baa2d6f2582b66... (64자리 16진수)

형태 2 (Encoding 키):

jXxBjM3%2FYvZ8Hk...%2B...%3D%3D (100자 이상, %2F 포함)

발급받은 일반 인증키(Encoding) 붙여넣기

💡 저장을 누르면 키가 포함된 새 파일이 자동 다운로드됩니다.
다운로드된 index.html로 교체하면 다른 사용자도 날씨를 볼 수 있습니다.

키를 붙여넣으면 형식을 자동으로 확인합니다.
